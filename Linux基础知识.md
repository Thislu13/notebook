## 1.2 内存查看

* 查看当前目录所在分区

  ```
  df -h .
  ```

  `df -h` 会显示当前目录所在的文件系统以及使用情况，`-h` 选项表示人类可读的格式（例如，显示为 GB 或 MB）

* 显示所有磁盘分区

```
lsblk
```

这个命令会列出系统中所有的磁盘和分区结构，还会显示每个分区的挂载点。

`-l`列出磁盘的详细信息
